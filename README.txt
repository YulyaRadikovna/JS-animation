Js-анимация посредством библиотеки GreenSock.

Вёрстка одноэкранного сайта (без адаптивной версии).

С помощью инструментария библиотеки GreenSock реализована анимация загрузки сайта и открытия бургер-меню.
При закрытии меню анимация с нуля не прописывается — для этого используется метод reverse().
